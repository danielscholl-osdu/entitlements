# Bootstrap

Below you can find required variables to bootstrap Entitlements Service on GCP and on-prem. They should be passed as environment variables.

## GCP usage

```bash
  DATA_PARTITION_ID
  ENTITLEMENTS_HOST
  ADMIN_USER_EMAIL
  PROJECT_ID
```

## On-prem usage

```bash
  DATA_PARTITION_ID
  ENTITLEMENTS_HOST
  OPENID_PROVIDER_URL
  OPENID_PROVIDER_CLIENT_ID
  OPENID_PROVIDER_CLIENT_SECRET
  ONPREM_ENABLED
```
